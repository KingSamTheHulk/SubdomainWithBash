Use this command:
cat subdomains.txt | ./subdomainlist.sh ....

In order to get the subdomain you can modify your subdomain.txt.
for example 
cat subdomains.txt | ./subdomainlist.sh yahoo.com 

Don't forget to:
chmod +x subs.sh
chmod +x subdomainlist.sh

The result will be:


test.yahoo.com
dev.yahoo.com
www.yahoo.com
m.yahoo.com
forums.yahoo.com
blog.yahoo.com
shop.yahoo.com
sports.yahoo.com
news.yahoo.com
mail.yahoo.com
